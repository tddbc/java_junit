TDDBC for Java with JUnit
====================================

これは、TDDBCのJava向けJUnitプロジェクトです。

## gradleを使う場合
### インストール
```bash
# for. Windows
./gradlew.bat

# for. Mac, Unix
./gradlew

# for. Mac
brew install gradle
```

### プロジェクトの変換
```bash
# for. IntelliJ IDEA
gradle idea

# for. Eclipse
gradle eclipse
```

### テストの実行
```bash
gradle test
```

## mavenを使う場合
### インストール
```bash
# for. Mac
brew install maven
```

### プロジェクトの変換
```bash
# for. IntelliJ IDEA
mvn idea:idea

# for. Eclipse
mvn eclipse:eclipse
```

### テストの実行
```bash
mvn test
```
